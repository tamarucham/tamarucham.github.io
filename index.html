3
Create a new repository in GitHub

Name it '<username>.github.io' (replace <username> with your GitHub username)

4
Create a file in the repository named 'index.html' and hit the edit button to edit in the browser

This will be your single webpage file.

5
Copy the content from Lovable into 'index.html' (it will replace the preview when you click "code")

Paste all the HTML code that Lovable generated for you into the 'index.html' file you created in GitHub.

6
Commit the changes with the green "Commit Changes" button

Save your changes and GitHub will automatically deploy your site. If asked, commit to the 'main' branch and not a new one.

7
You're done!

Browse to <your GitHub username>.github.io to see your website.

Go!
